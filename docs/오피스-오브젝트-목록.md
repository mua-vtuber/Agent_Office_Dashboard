## 제작 참고사항

- 모든 오브젝트는 충돌(collision) 영역으로 사용됨 (에이전트가 통과 불가)
- SVG viewBox 비율은 배치도 이미지 비율 참고하여 자유롭게 설정
- 오피스 캔버스에 PixiJS로 렌더링 예정 → SVG 단순 구조 권장
- 색상 톤: 배치도 기준 (베이지 책상, 녹색 캐비닛, 회색 가전)
